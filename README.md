# SYSU-Food-Picker (康乐园美食审判台)

基于 HarmonyOS ArkTS 开发的趣味随机美食抽取应用。

## 🌟 核心功能
- **分类均衡随机**：确保四大分类（食堂、外卖、出去吃、泡面）概率相等。
- **加权权重算法**：按 5:4:3:2:1 比例，评分越高的店被选中的几率越大。
- **智能 Buff 系统**：
  - 抽到“夯”店：建议省钱，拒绝奶茶。
  - 抽到“拉完了”店：提供避雷补偿（如加个蛋）。
- **中大南校园定制**：预设松涛、逸仙、下渡路等周边美食数据。

## 🛠️ 技术栈
- **Language**: ArkTS
- **Model**: Stage Model (API 9+)
- **IDE**: DevEco Studio
